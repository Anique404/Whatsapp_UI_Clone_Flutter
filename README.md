# WhatsApp UI Clone

A WhatsApp user interface clone built with Flutter.

## Features
- WhatsApp home screen with TabBar (Chats, Status, Calls)
- AppBar with icons
- Drawer menu
- Screen navigation

## How to Run
```bash
flutter pub get
flutter run
```

- Flutter
- Dart
